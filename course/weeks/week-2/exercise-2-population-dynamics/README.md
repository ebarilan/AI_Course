# Week 2 - Exercise 2: Population Dynamics

This folder contains the original exercise and a complete, student-friendly
Python solution.

## Files

- `original-exercise.pdf` - The original assignment PDF.
- `population_dynamics_solution.ipynb` - Runnable solution notebook for all four parts.
- `solution.md` - Short written summary of the verified results.

## Exercise summary

The exercise models algae, small fish, and large fish with a transition matrix.
It covers repeated matrix multiplication, conservation of total population,
eigenvalues and eigenvectors, long-term behavior, and different starting
populations.

Open the notebook in Google Colab from the course website and run the cells from
top to bottom.
