# Week 1 - Exercise 1: Linear Algebra

Use this folder for the files needed to understand, solve, and submit the exercise.

## Files

- `original-exercise.pdf` - The original assignment PDF.
- `part_1_notbook.ipynb` - Python notebook for word embeddings and distance calculations.
- `solution.md` - Written answers for matrix form, linearity, derivative matrix calculations, and interpretations.

## Exercise summary

The exercise covers word embeddings, solving a 3x3 linear system, proving matrix multiplication is linear, and working with a derivative matrix for polynomials of degree at most 4.
